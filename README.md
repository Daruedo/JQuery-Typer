# jQuery-typer

Teste sua velocidade de digitação.

Tecnologias:

 - Javascript;
 - Materialize.

Funcionalidades:

 - Contador de palavras;
 - Timer;
 - Troca de frase com método random;
 - Placar com sincronização AJAX com servidor.

Iniciando projeto:

```
cd .\server\

npm start
```
